Basic Computer Course

Start Your Digital Journey from the Basics

This course is designed for beginners who want to build confidence in using computers for study, office work, communication, and daily tasks.

Computer Fundamentals

Introduction to computer parts, operating system, desktop use, and basic settings.

Keyboard & Typing

Keyboard use, mouse control, typing practice, shortcuts, and basic input skills.

File Management

Create, save, copy, move, rename, organise, and manage files and folders.

MS Word Basics

Prepare letters, applications, tables, simple reports, formatting, and printing.

Internet Browsing

Search information, use websites, fill online forms, download files, and browse safely.

Email Basics

Create email, send messages, attach files, manage inbox, and use professional email.
Advanced Computer Course

The Advanced Computer Course is suitable for students, job seekers, office employees, and learners who want stronger computer skills for professional use.

Advanced MS Word

Formatting, tables, reports, letters, mail merge, and document design.

Advanced MS Excel

Formulas, charts, tables, data entry, basic analysis, and reporting.

PowerPoint

Professional slides, presentation design, animations, and layout.

Google Workspace

Google Docs, Sheets, Drive, Forms, Gmail, and cloud collaboration.

Office Operations

Printing, scanning, PDF handling, document conversion, and file sharing.

Basic Data Handling
