Web Development Modules

What You Will Learn

This course combines coding-based web development and WordPress website development so students can understand both technical and practical website building.

HTML

Learn website structure, headings, paragraphs, links, images, lists, forms, tables, semantic tags, and clean page layout.

CSS

Style websites using colours, fonts, spacing, layouts, responsive design, cards, buttons, animations, and modern UI design.

JavaScript

Add interactivity with menus, sliders, forms, popups, tabs, animations, events, and dynamic website features.

WordPress

Create websites using WordPress themes, pages, plugins, menus, forms, media, blog posts, and customisation tools.

Responsive Design

Build websites that work properly on mobile, tablet, laptop, and desktop screens.

Hosting & Deployment

Learn domain, hosting, GitHub Pages/basic deployment, file structure, and how to publish websites online.

Learning Pathway

Step-by-Step Web Development Learning Process

01

Understand Website Basics

Learn how websites work, what frontend development means, and how HTML, CSS, JavaScript, domain, and hosting connect together.

02

Create Webpage Structure with HTML

Build clean webpage sections using text, images, links, forms, buttons, lists, tables, and semantic HTML tags.

03

Design Modern Layouts with CSS

Apply colours, typography, spacing, grids, flexbox, cards, hover effects, gradients, and responsive design.

04

Add Interactivity with JavaScript

Learn dropdowns, sliders, tabs, popups, form interactions, animations, and basic dynamic behaviour.

05

Build Websites with WordPress

Create pages, menus, posts, plugins, forms, media galleries, and customised WordPress websites.

06

Publish and Manage Websites

Learn hosting, file management, deployment basics, website updates, backups, and basic maintenance.
